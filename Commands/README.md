# Commands

+ Linux
  - system
    - compress
    `tar -cvf targetname dirname`
    - uncompress
    `tar -xvf name`
    - find
    `find -name name`
  - tmux
    - connect to a session
    `tmux attach -t name`
    - create a session
    `tmux new -t name`
    - kill a session
    `tmux kill-session -t name`
    - leave session
    `ctrl b + d`
